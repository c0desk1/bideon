## ✨ Features

- 🔥 **Astro v5+** — Modern, fast static site builder
- ☁️ **Cloudflare Workers** — Deployed to the edge
- 📰 **Blogger CMS Integration** — Pulls content dynamically from Blogger API
- 🎨 **TailwindCSS v4.1** — Utility-first styling
- 🧩 **Componentized Layout** — Modular with clean codebase
- 📈 **SEO & OG Optimization** — Pre-rendered meta & OG tags
- 🔐 **Turnstile CAPTCHA** — Secure comments with Cloudflare Turnstile
- 💬 **Comment System** — Integrated, minimal, anti-spam
- 🔗 **Social Sharing** — OpenGraph-ready, perfect for social media
